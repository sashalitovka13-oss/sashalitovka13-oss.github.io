Туристичне Закарпаття — готовий шаблон дипломного сайту

Структура:
- index.html — головна сторінка
- style.css — стилі
- script.js — інтерактивність (тема, мова, фільтри, карта, анімації)
- assets/icons/logo-zakarpattia.svg — новий логотип
- assets/images/*.svg — локальні ілюстрації для секцій і карток

Що реалізовано:
- повністю нова структура коду
- без контактів, новин, входу, реєстрації, техпідтримки
- лише напрямок Закарпаття
- сучасний hero-блок
- спокійні анімації появи
- кнопка темної теми
- кнопка перемикання українська/англійська
- фільтрація за районом, типом місця та ключовим словом
- інтерактивна карта Leaflet на головній
- адаптація під телефон
- короткий footer-disclaimer

Як запустити:
1. Відкрий файл index.html у браузері
або
2. Запусти через локальний сервер, наприклад VS Code Live Server

Примітка:
Карта використовує Leaflet та OpenStreetMap через CDN, тому для коректної роботи потрібен інтернет.
